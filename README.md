# BongoCat

一款跨平台互动桌宠，为桌面增添乐趣！

## 项目起源

本项目基于 [BongoCat](https://github.com/ayangweb/BongoCat) 改编，继承了其可爱的猫咪互动功能，并针对特定需求进行了定制化开发。

### 原版 BongoCat 简介

BongoCat 是一款跨平台互动桌宠，最初灵感来源于 [Bongo-Cat-Mver](https://github.com/MMmmmoko/Bongo-Cat-Mver)。原版特点：

- 适配 macOS、Windows 和 Linux(x11)
- 根据键盘、鼠标或手柄的操作，同步对应的动作
- 支持导入自定义模型
- 完全开源，代码公开透明

## 记忆系统

本项目的记忆功能参考了 [LifeBook](https://github.com/Anson-Trio/LifeBook) 的设计理念，为桌宠赋予了长期记忆能力，使其能够记住与用户的互动历史。

## 功能特性

- 跨平台支持（Windows/macOS/Linux）
- 键盘/鼠标操作同步互动
- 自定义模型支持
- 记忆系统
- 完全开源

## 技术栈

- 前端：HTML/CSS/JavaScript
- 后端：Tauri
