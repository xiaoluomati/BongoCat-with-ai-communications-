## 二、情感标签枚举

根据 `src/utils/emotion.ts` 中 `EMOTION_MAP`，系统支持 **6 种情感**：

| 情感关键词 | 对应音频文件 | 说明 |
|-----------|-------------|------|
| 高兴 / 开心 | 高兴.wav | 积极正面情感 |
| 伤心 / 难过 | 伤心.wav | 悲伤情感 |
| 生气 / 愤怒 | 生气.wav | 负面激动情感 |
| 害怕 / 恐惧 | 害怕.wav | 恐惧情感 |
| 平静 / 淡定 | 平静.wav | 中性平静情感 |
| 嫌弃 / 厌恶 | 嫌弃.wav | 负面嫌弃情感 |

**标签格式**: `<emo>情感</emo>`，例如 `<emo>高兴</emo>`

### 3.3 IndexTTS 服务准备

#### 方式一：Docker 部署（推荐）

```bash
# 拉取镜像
docker pull indexai/index-tts-server:latest

# 运行容器（端口 9880:9890）
docker run -p 9880:9890 --gpus all indexai/index-tts-server:latest
```

#### 方式二：Conda 部署

```bash
# 克隆仓库
git clone https://github.com/indexai/index-tts-server.git
cd index-tts-server

# 创建 conda 环境
conda create -n indextts python=3.10
conda activate indextts

# 安装依赖
pip install -r requirements.txt

# 运行
python app.py
```

#### 方式三：使用已运行的远程服务

在应用设置中配置远程 IndexTTS 服务地址：
```
http://<远程IP>:9880
```
